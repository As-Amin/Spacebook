To create new Git repo:
git init (in directory)
git add .
git commit -m 'initial commit'
git remote add origin (link of repo)
git remote (check if setup correctly)
git push origin master

To Save changes, push to GitHub:
git add .
git commit -m 'message here'
git push origin master

To Delete files added before gitignore added:
git rm -r --cached .
git add .
git commit -m "Drop files from .gitignore"

Update repo on local with GitHub version: git pull (link of repo)
